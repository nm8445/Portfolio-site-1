# Portfolio Site

This is a personal portfolio website built with [Eleventy (11ty)](https://www.11ty.dev/) and [Nunjucks](https://mozilla.github.io/nunjucks/).  
It showcases my projects, skills, and experience.

## 🚀 Features
- Built with Eleventy (static site generator)
- Templating with Nunjucks
- Responsive design with HTML, CSS, and Bootstrap
- Modular layouts and reusable components

## 🛠️ Technologies Used
- Eleventy (11ty)
- Nunjucks
- Bootstrap
- CSS / HTML
- JavaScript
